# PWM-Generator-using-ARTY-A7-FPGA
PWM Generator on Arty A7 FPGA
Overview

This project implements a Pulse Width Modulation (PWM) Generator on the Digilent Arty A7 FPGA using Verilog HDL. The PWM signal is generated by comparing an 8-bit counter with a programmable duty cycle value. The design is suitable for applications such as LED brightness control, motor speed control, servo control, and digital power electronics.

Features
8-bit PWM resolution (256 duty cycle levels)
Programmable duty cycle (0–255)

Working Principle

The PWM generator uses an 8-bit counter that continuously increments with every clock cycle. The counter value is compared with the user-defined duty cycle value.

If counter < duty_cycle, the PWM output is HIGH.
Otherwise, the PWM output is LOW.

Changing the duty cycle changes the percentage of time the output remains HIGH during one PWM period.

How to Run
Open Vivado.
Create a new RTL project.
Add the Verilog source files.
Add the Arty A7 XDC constraints file.
Run Synthesis and Implementation.
Generate the bitstream.
Program the FPGA using the Hardware Manager.
Observe the PWM output on the assigned PMOD pin or LED.
